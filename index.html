Vendor Extensions

Formalized Add-ons for Swagger

What Vendor Extensions?

Exactly. The Open API Initiative Specification (the spec formally known as Swagger 2.0) mentions the support for vendor extensions to the spec throughout. Where are these vendor extensions? What additional expressions of an API do those vendor extensions allow? What tools support those extensions?

This project aims to answer those questions.

Extensions

One day this section will be a big, formal index of all the wonderful edge cases supported by vendor extensions. For now, there's just this short list.

  • x-json-schema: This extension adds key components of JSON Schema back into Swagger.
